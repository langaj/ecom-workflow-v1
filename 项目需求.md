# PROJECT_SPEC.md

# Ecom Workflow System V1

## 项目定位

内部使用的电商AI套图生产系统。

单租户。

公司内部使用。

不考虑SaaS。

不考虑多租户。

不考虑权限系统。

不考虑用户系统。

目标：

利用已有N8N智能体工作流完成商品套图生产。

系统只负责：

* 任务创建
* 任务管理
* 数据存储
* 文件存储
* 工作流调度
* 状态跟踪

AI推理与图片生成全部由N8N完成。
